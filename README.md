MaxWalker
=========

Standalone Browser based time tracker/counter for tasks

Installation
------------
```bash
npm install -g yo grunt-cli bower # Might need sudo
git clone git://github.com/woodyrew/MaxWalker.git
cd MaxWalker
npm install && bower install
grunt server
```
